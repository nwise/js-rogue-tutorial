Controls:

  • Use Arrow Keys, NumPad, or vim keys to Move
  • v: View Log
  • 5 or .(period): Wait
  • g: Pick up item
  • i: Inventory (press letter for inventory item to select)
  • c: Character Screen
  • s: Save
  • >: Descend when stairs(>)
  • Use mouse to see info or select target of ranged attack
